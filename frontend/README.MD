# Photo Slideshow (Frontend)

A **React + Vite + TypeScript** frontend scaffold for the photo slideshow project.

## 🚀 Development

### From VS Code (recommended)
1. Open this folder in VS Code.
2. When prompted, reopen in the **Dev Container**.
3. Once the container is ready, the dependencies will be installed automatically.
4. Run the dev server:

```bash
npm run dev
```

Then open the displayed URL (typically `http://localhost:5173`).

### Locally (without dev container)

```bash
npm install
npm run dev
```

## 🧱 Project structure

- `src/` – React source files
- `index.html` – Vite entry
- `.devcontainer/` – VS Code dev container configuration
